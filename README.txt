介绍：
提供拉伸和降噪两个功能，适用于为非线性处理前期节约时间。
如果你是小白，也可以把他当作一键出图的玩具。

首次使用：
安装python解释器到runtime内。双击install_require.bat, 运行成功后可以删除install_require.bat。
接着参考使用步骤。

使用：
1.  点击‘启动鱼香肉丝.bat’启动程序。
2.  点击选择文件——处理
3.  输出图在output文件夹。

注意事项：
1.  目前只支持TIFF（.tif）图像（浮点图像或16bit整形图像，不支持其它bit位的整形图像）。
2.  强烈建议先使用siril或者pi做完白平衡和DBE后, 保存为16bit整形TIFF，然后传入该程序处理。
3.  建议不要拿来处理大面积的目标
4.  如果出现了error，在任务管理器里结束COM Surrogate 进程，重新运行即可。
